# Sanah Tatla Portfolio

A responsive student portfolio built with React and Vite.

## Project structure

```text
src/
  assets/       Images and resume files
  components/   Shared interface components
  data/         Portfolio content
  sections/     Home, about, projects, and resume sections
  styles/       Global responsive styles
```

## Development

```bash
npm install
npm run dev
```

Run `npm run lint` to check the code and `npm run build` to create a
production build.
